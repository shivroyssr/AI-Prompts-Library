## 📖 About

**AI-Prompts-Library** is a growing collection of carefully designed prompts for Large Language Models (LLMs). It helps users improve productivity, research, learning, and professional writing using AI. 

A collection of AI prompts for ChatGPT, Claude, Gemini, and professional writing.

This repository is suitable for:

- 🤖 AI Prompt Engineers
- 📊 AI Data Evaluators
- 📝 AI Annotators
- 🔬 Researchers
